build (handlers)
Builds LPeg grammar table.

The grammar expects 3 arguments to be passed into match function: the subject (text being parsed) and two empty tables. The first table is always passed into on_match_rule handler and may be used to store state information during parsing. The second table is used internally for parsing here-documents.

Parameters:

  • handlers optional {[string]=function,...} Table of custom handlers.

Returns:

    table LPeg grammar table.

Usage:

    local parser = lpeg.P(grammar.build())
parser:match(subject, 1, subject, {}, {})
