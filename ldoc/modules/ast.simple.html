__call (subject, opts)

Creates a function that transforms positional captures produced by the shell parser into AST nodes.

Options:

  • comments: Include comments.
  • loc: Include nodes' location as start and end absolute position in the subject.
  • loc2: Include nodes' location as start and end row, column and absolute position in the subject.
  • source: Include raw source in all nodes except the root node.

Parameters:

  • subject string The parsed input.
  • opts {[string]=bool,...} The options map.

Returns:

    create_node
