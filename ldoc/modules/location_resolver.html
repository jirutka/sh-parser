index_lines_pos (text)
Returns a list with starting positions of lines in the given text string. Note: only LF (\n) is considered as a line separator.

Parameters:

  • text string The string to index.

Returns:

    {int,...} A list of numbers, where index corresponds to a line number and value to a position inside text where the line starts.
find_nearest_lower (list, value, start_idx, end_idx)
Returns an index of item in the given sorted list that is largest from the items smaller or equal to the specified numeric value. This implementation uses modified binary search algorithm.

Parameters:

  • list {number,...} A sorted list of numbers.
  • value number The reference value.
  • start_idx optional int (default is 1)
  • end_idx optional int (default is #list)

Returns:

    number or nil An index of the item closest to the value, or nil if not found.
