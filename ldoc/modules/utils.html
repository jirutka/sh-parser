partial (fn, ...)
Partial application. Takes a function fn and arguments, and returns a function fn2. When applied, fn2 returns the result of applying fn to the arguments provided initially followed by the arguments provided to fn2.

Parameters:

  • fn function
  • ... Arguments to pass to the fn.

Returns:

    function A partially applied function.
